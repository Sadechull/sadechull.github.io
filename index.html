Machine Learing(2017 Fall) 学习笔记（一）

发表于 2018-11-03 | 分类于 Machine Learing(2017 Fall) 学习笔记

概念

  • f: model
  • y: label
  • x: feature
  • w: weight
  • b: bias

Loss Function

  • Gradient descent 梯度下降法
  • Local minimal/Global
  • Regression 回归
  • Regularization 归一化：加上一个权重依赖项，用于平滑损失函数

Where does the error come from?

  • 错误来源
  1. bias 偏差，瞄不准
  2. variance 方差，射击散布
  • 简单的模型bias大，variance小，容易欠拟合
  • 复杂的模型bias小，variance大，容易过拟合
  • 交叉验证：将训练集分为训练集(training set)和验证集(validation set)，训练到基线(baseline)以下后，在布署到公共集上

Gradient Descent

  • N维梯度的概念：Loss函数等高线的法线方向
  • Learning rate 学习速率/步长，步长要合适
  • 可变步长：1/t衰减 $\eta^t=\sqrt{t+1}$
  • Adagrad: $$w^{t+1}=w^{t}-\frac{\eta^t}{\sigma^t}g^t$$
  • 随机梯度下降：一次只训练一个数据
  • Feature Scaling: 针对步长不同的x调整特征为均值为零方差为1的序列
  • 数学推导：多元泰勒展开
